该应用程序可以不依赖C++ Builder环境来运行,双击即可运行。
也可以在C++Builder6.0中打开APITools.bpr文件，直接编译运行即可。
该程序是用来辅助APITestProj程序来运行的。
运行该程序可以看到APITestProj程序都捕捉到了什么窗口，而且也可以清除掉这些捕捉下来的窗口。同时设置APITestProj程序下一次运行要不要隐藏在后台，禁止那些被捕捉窗口的运行。
注意：该程序不要在光盘下运行，可以将其拷贝到硬盘上。因为它在关闭时会生成一个INI文件，而光盘是不可写的，所以会出错.

(注：有时在有的机器上运行时可能会出现一个“Invalid resourec file:....”的警告。
 不用管它，单击“OK”按钮即可，程序会在编译中重新生成这个文件）
